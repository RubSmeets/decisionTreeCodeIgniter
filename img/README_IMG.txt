The images folder contains image resources that are used by our website.

The root of the directory contains the images that are used throughout the website as background images or logos.

The img/logos/ directory contains the logo images of the contributed frameworks. The logo images that are submitted through
the webinterface by users and admins are stored here.
NOTE: please do not delete or modify the files in img/logos folder unless you know what you are doing. Changing/removing files
in these folders can result in incorrectly displayed content on the website.
